## Smart Home Provider API

https://t3chfest.uc3m.es/2019/programa/integrating-iot-projects-with-google-assistant-smart-home-api/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# incremental rebuild (webpack)
$ npm run webpack
$ npm run start:hmr

# production mode
$ npm run start:prod
```
